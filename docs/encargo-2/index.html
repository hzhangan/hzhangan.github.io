Encargo 2 - Silla Fresia

En este mecanizado de producto ocuparemos de ejemplo la Silla Fresia:

Mecanizado CNC

Paso 1 - Disign:

Definimos nuestro área de trabajo con un sketch de 1220mm x 1220mm y 15mm de grsor que es equivalente a la mitad de una plancha de terciado estandar. Luego, desglozamos la silla copiando cada uno de sus piezas y las alineamos con el plano para asegurar que vamos a poder acomodar las piezas dentro de la plancha en un mismo espacio de trabajo. Es importante que el calado de la pieza siempre quede mirando hacia arriba.

Una vez que entendemos cómo mover cada pieza para alinearlo con la plancha de terciado, realizaremos la misma dinámica con cada una de las partes de la silla para obtener el siguiente resultado:

Tutorial mecanizado CNC

Paso 2 - Manufacture:

Una vez que tengamos el espacio configurado con todas las piezas que queremos mecanizar en la plancha de terciado, cambiaremos de modo Diseño a Manufactura para acceder a las herramientas 2D.

Primero configuraremos las indicaciones necesarias para el mecanizado. En esta ocación ocuparemos la máquina Autodesk CNC de 3 Axis. Definimos nuestro modelo de orientación, seleccionamos la plancha como sólido sobre el cual trabajaremos y debemos definir cuáles serán las piezas que queremos cortar.

La primera operación que realizaremos es un 2D Contour con una fresa Flat end mill de 6mm, herramienta de corte que ocuparemos para este trabajo. Algo importante que debemos considerar es que una fresa de 6mm no puede perforar una plancha de 15mm de una sola pasada porque se puede desgastar o no realizará un trabajo eficiente. Por lo tanto, en Multiple Depths definiremos un Maximum Roughing Stepdown de 3mm para que la fresa realice el calado en 5 pasadas en vez de 1.

La segunda operación la realizaremos con la función 2D Pocket para trabajar las caras de los calados de las piezas. Mantendremos la misma lógica que usamos para configurar la operación anterior.

Por último, ocuparemos nuevamente el 2D Contour para definir el área exterior de cada parte de la silla. En este paso, además de definir Multiple Depths, debemos tener en cuenta que la pieza no debe quedar suelta ya que la fresa podría romper la pieza antes de terminar el trabajo. Para esta situación usaremos también los Tabs que definen pequeñas áreas en donde la fresa no va a generar un mecanizado. Basta con que tengamos algunos Tabs para sujetar la pieza.

Trazado

Aquí realizaremos nuevamente un sketch, esta vez de una plancha de terciado con 9mm y definiremos el contorno exterior del asiento con la función 2D Contour como lo hicimos anteriormente. La parte del asiento tiene varios trazados en la parte inferior de la pieza, estos recortes sirven para generar la flexibilidad del material. Para poder generar estos trazados ocuparemos la función Trace para definir cada linea de recorte. Le pediremos a la fresa que haga cortes con 4.5mm de profunidad.
